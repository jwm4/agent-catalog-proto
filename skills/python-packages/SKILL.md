# Python Language Reference

## Package Management

Use `addPackage("pip", [...])` to install Python packages.

## Common Packages by Use Case

- **Web:** flask, django, fastapi, uvicorn
- **Data science:** numpy, pandas, scikit-learn, matplotlib
- **ML/AI:** torch, transformers, langchain
- **Testing:** pytest, pytest-cov
- **Linting:** ruff, mypy
