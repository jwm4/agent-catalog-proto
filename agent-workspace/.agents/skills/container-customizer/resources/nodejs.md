# Node.js / TypeScript Language Reference

## Installation

Node.js is NOT in the OpenCode base image by default.
Install via: `addPackage("microdnf", ["nodejs", "npm"])`

## Common Packages by Use Case

- **Runtime:** tsx, ts-node
- **Build:** typescript, vite, esbuild
- **Testing:** vitest, jest, playwright
- **Linting:** eslint, prettier
- **Package managers:** pnpm, yarn
